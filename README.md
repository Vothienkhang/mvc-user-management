# MVC User Management

## Technologies
- Java Servlet
- JSP
- JDBC
- MySQL
- Bootstrap 5

## Database setup
1. Create database: `user_management`
2. Run file: `user_management.sql`

## Default DB config in code
- URL: `jdbc:mysql://localhost:3306/user_management`
- Username: `root`
- Password: `123456`

If your MySQL password is different, please update it in `UserDAO`.
